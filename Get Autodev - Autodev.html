Downloading the upgrade is easy

  • Click Get started to install Windows 8 Upgrade Assistant. Depending on your Internet connection, the installation might take several minutes.

  • When prompted by your browser, open, save, or run Upgrade Assistant. You must run Upgrade Assistant to purchase Windows 8 or Windows 8 Pro online, but you can also run it without purchasing.

  • Follow the instructions in Upgrade Assistant to find out if your PC can upgrade to Windows 8 or Windows 8 Pro, and then follow the steps to purchase, download, and install. To learn more, see How to download the Windows 8 upgrade . For more detailed information about upgrading, see Upgrade to Windows 8.

System requirements

To install the Windows 8 or Windows 8 Pro upgrade, your PC must be running Windows XP SP3, Windows Vista, Windows 7, Windows 8 Consumer Preview, or Windows 8 Release Preview. Learn how to get Windows 8 on a PC not running a previous version of Windows .

If your PC is running Windows 7, Windows 8 Consumer Preview, or Windows 8 Release Preview, your files, apps, and settings will easily transfer to Windows 8 or Windows 8 Pro. If your PC is running Windows XP or Windows Vista, you'll need to reinstall your apps. Some features such as touch might require a new PC.
